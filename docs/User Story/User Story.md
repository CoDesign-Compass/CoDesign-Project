**User Side**

1. As a user, I want to create a simple profile with my name so that I can start the survey.
2. As a user, I want to select up to 20 tags that describe me (e.g., dog person, student, sport lover) so that the survey feels personalized.
3. As a user, I want to add my own custom tags so that I can represent myself more accurately.
4. As a user, I want to answer whether I think each issue is a problem or not so that my opinion is captured.
5. As a user, I want to explain why I think something is (or isn’t) an issue so that my perspective is understood.
6. As a user, I want to go through a chain of follow-up “why” questions so that I can reflect deeply on my reasoning.
7. As a user, I want to provide “how” answers for solutions so that my ideas for improvement are included.
8. As a user, I want to submit my email at the end so that I can receive a $10 Coles voucher.
9. As a user, I want to choose whether I am willing to fill out similar surveys in the future so that I have control over my participation.
10. As a user, I want to decide whether to save my profile and create an account (email + password) so that I can reuse it next time.



**Administrator Side**

1. As an administrator, I want to add new survey questions so that the content stays relevant.
2. As an administrator, I want to remove outdated survey questions so that the survey remains focused.

3. As an administrator, I want to edit existing survey questions so that I can refine how they are asked.

4. As an administrator, I want to view a dashboard with all user responses so that I can analyze the collected data.

5. As an administrator, I want to use AI to generate summaries of answers for each question so that I can quickly see key insights.

6. As an administrator, I want to see statistics (e.g., percentage of “yes” vs “no”) for each question so that I can track trends.

7. As an administrator, I want to filter results by user tags (e.g., student, dog person) so that I can analyze different user groups.

8. As an administrator, I want to export response data so that I can use it for external analysis.

9. As an administrator, I want to track how many users completed the survey so that I can measure engagement.

10. As an administrator, I want to see how many users agreed to future surveys so that I can plan upcoming research.
